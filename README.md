# How To Install

## First you need to install dependent packages
`pip install colorama keyboard psutil`

# How To Use
## How To Start The Script

### Open cmd, and enter the next text:
`python fwu.py`

Profit!
## How To Open Settings

### Open cmd, and enter the next text:
`python fwu.py -s`

Customize to your taste
